# Peek-a-Hide - Device Updates

Device update bundles for the **Peek-a-Hide** reptile hide camera system by [AuDHDly Reptile             
Studio](https://reptile-studio.audhdly.com).
                                                                                                         
This repository contains no source code, only release artifacts. The latest release is always at the top 
of the [Releases](../../releases) page.
                                                                                                         
---             

## For Customers — Installing an Update                                                                  
 
You do not need a GitHub account to install an update. Your device downloads the update directly — there 
is no file to download or upload.
                                                                                                         
### What you'll need                                                                                     
 
- Your Peek-a-Hide connected to your home Wi-Fi (with internet access)                                   
- The customer password that came with your device
- A browser on the same Wi-Fi network as the device
                                                                                                   
See the [Updating Your Device instructions](https://peek-a-hide.audhdly.com/user-guide?step=12) in the Peek-a-Hide User Guide for the rest of the instructions.
